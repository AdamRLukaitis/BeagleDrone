Beagledrone

An open source drone OS for the BeagleBone

Download this project as a .zip file Download this project as a tar.gz file

The BeagleDrone aims to be a software platform for developing 'bare metal' drones running on the BeagleBone, and possibly other System-on-Chip (SoC) devices.

Documentation

Developer documentation can be found on our wiki.

Contributing

If you are interested in contributing to BeagleDrone please have a look at the Contributing Guide in the developer wiki.

Open tickets can be found on our issue tracker.

Feel free to create tickets if you run into problems or would like to request a feature.

Please join us in IRC (#beagledrone on freenode) to discuss an issue, chat about robots, or just say Hello!
